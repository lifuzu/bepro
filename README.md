### Backend
Wordpress

#### Steps:
`$ docker-compose up -d`

Nodejs/Expressjs - https://expressjs.com/en/starter/generator.html

#### Steps:
`$ npm i express-generator -g`
`$ express --view=pug node`
`$ npm install` - install dependencies
`$ DEBUG=node:* npm start` - run the app
